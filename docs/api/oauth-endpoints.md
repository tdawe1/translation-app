# OAuth Endpoints

Endpoints for managing social login and account linking with third-party providers.

## List Available Providers
Retrieve a list of supported OAuth providers and their current status for the user.

- **URL**: `/oauth/providers`
- **Method**: `GET`
- **Auth Required**: No (Can be called while logged in to see linked accounts)

### Success Response
- **Code**: `200 OK`
- **Body**:
```json
{
  "data": [
    {
      "provider": "google",
      "enabled": true,
      "linked": true
    },
    {
      "provider": "github",
      "enabled": true,
      "linked": false
    }
  ]
}
```

---

## Authorize Provider
Get the redirect URL for an OAuth provider to begin the authentication flow.

- **URL**: `/oauth/authorize`
- **Method**: `GET`
- **Auth Required**: No
- **Parameters**: `provider` (required: `google`, `github`)

### Success Response
- **Code**: `200 OK`
- **Body**:
```json
{
  "auth_url": "https://accounts.google.com/o/oauth2/v2/auth?..."
}
```

---

## OAuth Callback
The endpoint called by the OAuth provider after successful user authentication.

- **URL**: `/oauth/{provider}/callback`
- **Method**: `GET`
- **Auth Required**: No

### Query Parameters
- `code`: The authorization code from the provider.
- `state`: The CSRF state token generated by the server.

### Success Response
- **Code**: `302 Found`
- **Redirects to**: Your dashboard URL (configured via `OAUTH_REDIRECT_URL`).
- **Note**: Sets the session cookies upon success.

---

## Unlink Provider
Disconnect an OAuth provider from the current user account.

- **URL**: `/oauth/unlink`
- **Method**: `POST`
- **Auth Required**: Yes

### Request Body
```json
{
  "provider": "github"
}
```

### Success Response
- **Code**: `204 No Content`

### Error Response (400)
- **Code**: `BAD_REQUEST`
- **Reason**: "Cannot unlink last authentication method." (User must have at least one password or one OAuth link).
