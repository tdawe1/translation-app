# Changelog

All notable changes to the GengoWatcher SaaS project will be documented in this file.

## [1.0.0] - 2026-01-06

### Added
- **Authentication**: Email/Password, Magic Link, and Google/GitHub OAuth.
- **Watcher System**: Multi-tenant RSS and WebSocket job monitoring.
- **Dashboard**: Real-time job alert interface using WebSockets and Redis Pub/Sub.
- **Billing**: LemonSqueezy integration for Pro and Enterprise subscriptions.
- **API**: Comprehensive RESTful API with OpenAPI/Swagger documentation.
- **Documentation**: Professional multi-section documentation suite.

---

## [0.5.0] - 2025-12-15

### Added
- **Backend**: Core Go API using Fiber framework.
- **Database**: Initial schema with PostgreSQL 17 and Alembic migrations.
- **Frontend**: Next.js 16 scaffolding with Tailwind CSS 4.x.
- **DevOps**: Docker Compose configuration for local development.

---

## [0.1.0] - 2025-11-01

### Added
- **Project Concept**: Transformation of GengoWatcher from local tool to SaaS.
- **Architecture**: Design of the multi-tenant watcher manager system.
